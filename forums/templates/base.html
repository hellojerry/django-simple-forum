BBCode Function Options Examples
B creates bold text [b]bold text[/b]
i creates italic text [i]italics text[/i]
u creates underlined text [u]underlined text[/u]
s strikes through text [s]strike text[/s]
list creates an unordered list 1: ordered list, A: ordered list (alphabetic) [list][*]one[*]two[/list], [list=1][*]one[*]two[/list]
* creates a list item [list][*]one[*]two[/list]
code retains all formatting [code][b]example[/b][/code]
quote creates a blockquote [quote]quoted string[/quote]
center creates a centered block [center]example[/center]
color changes text color [color=red]red text[/color], [color=#FFFFFF]white text[/color]
url makes a url [url]http://example.com[/url]
img makes an image [img]http://xyz.com/logo.png[/img]
